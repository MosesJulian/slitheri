# Slitheri
Just a simple snake game

# Requirements
- pnpm OR npm OR yarn

# Installation
- Clone the repository
```bash
$ git clone git@github.com:MosesJulian/slitheri.git
$ cd slitheri
$ pnpm install
OR
$ yarn install
OR
$ npm install
```
- Run the application
```bash
$ pnpm dev
OR
$ yarn dev
OR
$ npm run dev
```
